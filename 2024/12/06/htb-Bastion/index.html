0x01:Shell as L4mpje

在普通情况下，vhd文件是有被限制读取的。但是由于我们是将vhd驱动挂载，因此我们拥有全部的访问权限，可以访问注册表文件。

SAM 文件：SAM (Security Accounts Manager) 文件是 Windows 系统中用于存储用户账户信息和加密密码哈希的位置。这个文件通常位于：C:\Windows\System32\config\SAM
Security文件：存储了 本地安全策略和与用户身份验证、访问控制列表（ACLs）、权限管理等相关的信息。这个文件位于C:\Windows\System32\config\Security
NTLM 哈希格式如下，其中aad3b435b51404eeaad3b435b51404ee 是 LM 哈希，31d6cfe0d16ae931b73c59d7e0c089c0 是 NTLM 哈希。

NTLM 和 Kerberos都是 Windows 系统中用于身份验证的协议。Kerberos 是一个现代的、功能更强的协议，而 NTLM 主要是老旧系统中使用的协议。

使用Secretdump可以将SAM和Security中的Hash读取出来

https://crackstation.net/ 可以对NTLM Hash进行破解，破解结果如下L4mpje:bureaulampje
也可以使用hashcat进行破解，-m=1000时指的是破解NTLM Hash

使用此用户凭据ssh连接
