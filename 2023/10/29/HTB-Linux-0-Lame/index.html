2-Summary

Samba

Samba is a protocol for sharing files, printers or other resources. Two common port used:

  • 139 commonly used in early samba
  • 445 commonly used in common network environment

Samba USERNAME Map Script

Before authentication, USER_NAME is mapped and parsed. User_name will be executed as a command.

Vsftpd

vsftpd is a usual FTPserver based in unix. It used in port 22.
